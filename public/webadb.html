📱 WebADB Phone Mirror

Mirror HP Android langsung di browser - Tidak perlu install apapun!

🚀 Cara Pakai:

  1. Aktifkan USB Debugging di HP Android
  2. Colok HP ke laptop/PC via kabel USB
  3. Klik tombol "Connect via USB"
  4. Browser akan minta izin akses USB - klik Allow
  5. Pilih perangkat HP Anda dari popup
  6. Klik "Start Screen Capture" untuk mulai mirroring
Siap untuk memulai - Klik "Connect via USB"
