Phone Mirror

Mirror HP ke PC/Laptop

Langkah-langkah:

  1. Jalankan server di laptop: npm run local
  2. Masukkan IP laptop (contoh: http://192.168.1.5:3001)
  3. Sambungkan HP ke laptop via USB
  4. Aktifkan USB Debugging di HP
  5. Klik "Scan Perangkat" lalu "Connect"
